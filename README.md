## 🎈 프로젝트 소개

**Open Mind**는 익명성을 보장하며 서로 다른 사람들과 소통하기 위한 플랫폼입니다.

CRUD 기능을 구현한 **Open Mind**로 다른 사람들과 익명으로 소통해보세요!
